## 🚀 Key Features
* **User Authentication:** Secure Login, Logout, and Registration.
* **Expense Tracker:** Add and categorize spending in real-time.
* **Finn AI Advisor:** Receives logic-based financial insights and warnings.
* **Modern Dashboard:** Built with a professional, responsive UI using Bootstrap 5.

## 🛠️ Tech Stack
* **Backend:** Python (Flask)
* **Database:** SQLite
* **Frontend:** HTML5, CSS3, Bootstrap 5
* **Logic:** Finn AI Algorithm
